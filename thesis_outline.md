# Rust 非正常控制流下的内存安全问题研究
Research on Memory Safety Issues in Rust Under Abnormal Control Flow

## 摘要与关键词
- 中文摘要（与 `abstract.md` 一致）
  - 背景：Safe Rust 默认内存安全；panic/unwind 下异常控制流与 Drop、unsafe、FFI 交织引入双重释放、UAF、跨边界 UB 等风险
  - 问题：panic/展开语义下分类标准不统一、评测不系统、工具能力边界不清
  - 方法：（1）触发点—资源状态—隔离边界三维度场景分类（S1–S4）及机理分析；（2）基于场景分类的静态分析评测框架，在统一用例集上评测 Rudra、MirChecker、FFIChecker
  - 结果：基础指标 + 分场景 Recall 等量化对比；S3 上 FFIChecker 相对通用工具的反差、工具间互补与短板（摘要中给出具体数值）
  - 贡献：第 3 章 场景分类与机制解释；第 4 章 评测框架、实证结论与可复现流程
- 关键词：Rust；panic；非正常控制流；Drop；unsafe；FFI；静态分析；能力边界；漏洞挖掘
